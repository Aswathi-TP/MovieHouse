MOVIEHOUSE
DESCRIPTION
This project is a Movie Ticket Booking System developed using Django. It allows users to browse movies, select seats and book tickets with a simple payment workflow.
FEATURES
1. User registration and login authentication
2. Browse available movies and show timings
3. Seat selection system
4. Ticket booking functionality
5. Basic payment process simulation
TECHNOLOGIES USED
1. Python
2. Django
3. SQLite
4. HTML
5. CSS
6. Bootsratp
7. JavaScript
KEY LEARNING
* Backend development using Django
* Handling user authentication
* Managing database operations
* Understanding real-worls booking workflows
